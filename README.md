# AWS S3 file upload progress (aws-s3-file-upload-progress)
AWS S3 file upload with progress bar 

## Feature 
1. File Upload 
2. Progress bar 

## Perquisite
- AWS account
- S3 bucket
- CORS enable for S3 bucket 
- IAM user with Secret key and access key
